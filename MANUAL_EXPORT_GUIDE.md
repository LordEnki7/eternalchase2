- `vite.config.ts` - Build configuration
- `tailwind.config.ts` - Cosmic theme setup
- `drizzle.config.ts` - Database migrations

### 3. Documentation Files
- `README.md` - Complete project overview
- `DEPLOYMENT.md` - Platform-specific deployment instructions
- `EXPORT_CHECKLIST.md` - Post-migration verification

## 📋 Files to EXCLUDE

**Do NOT copy these:**
- `node_modules/` - Will be regenerated by npm install
- `dist/` - Build output, regenerated on deployment
- `package-lock.json` - Platform-specific, regenerated
- `migrations/` - Auto-generated database files
- `.git/` - Git history (optional to exclude)
- `.replit` - Replit-specific configuration

## 🔧 Post-Export Setup

### 1. Initialize New Project
```bash
# Extract files to new directory
mkdir eternal-chase-new
cd eternal-chase-new
# Copy all files from backup

# Install dependencies
npm install
```

### 2. Environment Configuration
```bash
# Create environment file
cp .env.example .env

# Add your secrets:
DATABASE_URL=your-postgres-url
SESSION_SECRET=your-session-secret
STRIPE_SECRET_KEY=your-stripe-secret
VITE_STRIPE_PUBLIC_KEY=your-stripe-public
REPL_ID=your-replit-id
REPLIT_DOMAINS=your-domain.com
```

### 3. Database Setup
```bash
# Push schema to new database
npm run db:push
```

### 4. Test Deployment
```bash
# Development mode
npm run dev

# Production build
npm run build
```

## 📞 Support After Migration

### Common Issues
1. **Environment Variables**: Double-check all secrets are set correctly
2. **Database Connection**: Ensure DATABASE_URL format is correct
3. **Domain Configuration**: Update REPLIT_DOMAINS for new hosting
4. **Stripe Keys**: Verify test vs live keys match your intent

### Verification Steps
1. Landing page loads for non-authenticated users
2. Sign-in redirects to Replit correctly
3. Authenticated users see cosmic portal
4. Subscription modal and payment processing works
5. Premium content protection functions properly

### Required Services
- **PostgreSQL Database**: Any provider (Neon, Supabase, Railway, etc.)
- **Node.js Hosting**: Version 18+ required
